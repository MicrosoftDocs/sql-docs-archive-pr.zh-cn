  报表部件是自包含的报表项，存储在报表服务器上并可以包含在其他报表中。 使用报表生成器可以从报表部件库中浏览和选择部件，以添加到您的报表中。 使用报表设计器或报表生成器可以将报表部件保存在报表部件库中以供使用。 有关详细信息，请参阅 Web 上 microsoft.com 中的“报表部件（报表生成器和 SSRS）”  和“报表设计器中的报表部件 (SSRS)”  。
