# <a name="package-and-project-password"></a>包和项目密码
  可以使用 **“包密码”** 对话框为使用密码加密的包提供包密码。 如果包使用 **“使用密码加密敏感数据”** 或 **“使用密码加密所有数据”** 保护级别，则必须提供密码。  
  
## <a name="options"></a>选项  
 **密码**  
 输入密码。
