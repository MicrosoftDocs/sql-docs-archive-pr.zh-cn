# <a name="destination-assistant"></a>目标助手
  目标助手组件可以帮助创建目标组件和连接管理器。 该组件位于 SSIS 工具箱的 **“收藏夹”** 部分中。  
  
> [!NOTE]  
>  目标助手替换 Integration Services 连接项目和相应向导。  
  
## <a name="related-tasks"></a>Related Tasks  
 有关如何使用目标助手添加源和连接管理器的说明，请参阅 [Add a Destination using Destination Assistant](../add-a-destination-using-destination-assistant.md)。
