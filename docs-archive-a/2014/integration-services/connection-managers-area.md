# <a name="connection-managers-area"></a>连接管理器区域
  包使用连接管理器连接到文件、关系数据库和服务器等数据源。  
  
 使用 **设计器的** 连接管理器 [!INCLUDE[ssIS](../includes/ssis-md.md)] 区域可以添加、删除、修改、重命名以及复制和粘贴连接管理器。  
  
 右键单击此区域，然后在菜单上单击与要执行的任务相应的选项。
