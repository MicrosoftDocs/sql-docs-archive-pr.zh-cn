# <a name="save-reports-to-your-computer-report-builder"></a>将报表保存到您的计算机（报表生成器）
  在报表生成器中，可以将报表定义保存到您的计算机。 不过，将报表保存到您的计算机后，其他用户将不能查看这些报表，并且引用共享数据源或存储报表项（如图像和子报表）的报表可能不能在外部运行。 建议将报表保存到报表服务器或 SharePoint 站点。  
  
### <a name="to-save-a-report"></a>保存报表  
  
1.  从“报表生成器”按钮，单击 **“保存”**。 此时将打开 "**另存为**" _\<Report Item>_ 对话框。  
  
    > [!NOTE]  
    >  如果正在重新保存报表，会自动将其重新保存到以前的位置。 使用 **“另存为”** 选项可以更改位置。  
  
2.  （可选）单击 "**桌面**"、 **"我的文档**" 或**我的电脑**将报表保存到其中一个文件夹。  
  
3.  找到计算机上要保存该报表的位置。 在 **“名称”** 中，键入报表的名称。  
  
4.  在 **“项类型”** 中，选择要保存的报表项的类型。 报表的类型为 Reports(*.rdl)。  
  
### <a name="to-save-a-report-as-a-different-name"></a>将报表另存为不同名称  
  
1.  从 “报表生成器” 按钮，单击 **“另存为”**。 此时将打开 "**另存为**" _\<Report Item>_ 对话框。  
  
2.  找到计算机上要保存该报表的位置。  
  
3.  在 **“名称”** 中，键入报表的名称。 （可选）单击 "**桌面**"、 **"我的文档**" 或**我的电脑**将报表保存到其中一个文件夹。  
  
4.  在 **“项类型”** 中，选择要保存的报表项的类型。 报表的类型为 Reports(*.rdl)。
