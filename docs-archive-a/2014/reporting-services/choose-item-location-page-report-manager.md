# <a name="choose-item-location-page-report-manager"></a>“选择项位置”页（报表管理器）
  使用“选择项位置”页可以为新链接报表或新模型选择文件夹。 如果要为特定的用户组创建链接报表或模型，则最好将该项放在包含这组用户所使用的其他报表和模型的文件夹中。 必须选择一个已经存在并且有权对其添加内容的文件夹。  
  
## <a name="navigation"></a>导航  
 使用以下过程导航到用户界面 (UI) 中的这一位置。  
  
###### <a name="to-open-the-choose-item-location-page-for-a-report"></a>打开报表的“选择项位置”页  
  
1.  打开报表管理器，找到要添加链接报表的报表。  
  
2.  悬停在该报表之上，然后单击下拉箭头。  
  
3.  在下拉菜单中，执行以下步骤之一：  
  
    -   单击 **“创建链接报表”** 可以打开“新建链接报表”页。  
  
    -   单击 **“管理”** 以打开报表的“常规属性”页。 然后，单击 **“创建链接报表”** 以打开“新建链接报表”页。  
  
4.  在 **“常规”** 选项卡的属性页中，单击 **“更改位置”** 以打开“选择项位置”页。  
  
###### <a name="to-open-the-choose-item-location-page-for-a-model"></a>打开模型的“选择项位置”页  
  
1.  打开报表管理器，找到要添加模型的数据源。  
  
2.  悬停在该数据源之上，然后单击下拉箭头。  
  
3.  在下拉菜单中，执行以下步骤之一：  
  
    -   单击 **“生成报表模型”** 以打开“新建模型”页。  
  
    -   单击 **“管理”** 以打开数据源的“常规属性”页。 然后，单击 **“生成模型”** 以打开“新建模型”页。  
  
4.  在 **“常规”** 选项卡的属性页中，单击 **“更改位置”** 以打开“选择项位置”页。  
  
## <a name="options"></a>选项  
 **位置**  
 指定用来包含所创建项的文件夹的名称。 您可以键入全名，也可以使用树视图导航到想要使用的文件夹。  
  
 **树视图**  
 显示报表服务器命名空间的文件夹结构。 单击文件夹名称可以向 **“位置”** 字段中添加完整路径。  
  
 单击树视图中的展开 (+) 和折叠 (-) 图标可以打开和关闭文件夹，而不会向 **“位置”** 字段中添加文件夹名称。 若要向 **“位置”** 字段中添加文件夹名称，请单击文件夹名称。
