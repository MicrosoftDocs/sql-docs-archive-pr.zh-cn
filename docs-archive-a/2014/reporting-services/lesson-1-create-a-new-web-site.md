# <a name="lesson-1-create-a-new-web-site"></a>第 1 课：创建新网站
  在本课程中，你将学习如何使用 Visual C# 的 ASP.NET 网站模板创建新网站项目。  
  
### <a name="to-create-a-new-website"></a>创建新的网站  
  
1.  打开 Microsoft Visual Studio 2010 或 Microsoft Visual Studio 2012。  
  
2.  在“文件”菜单上，指向“新建”，然后选择“网站”    。  
  
3.  在 "**新建网站**" 对话框的 "**已安装的模板**" 窗格中，选择 " **Visual c #** "，然后选择 " **ASP.NET 网站**"。  
  
4.  在 "**位置**" 框中，指定一个项目目录，然后单击 **"确定"**。  
  
 随后将打开该网站项目。
