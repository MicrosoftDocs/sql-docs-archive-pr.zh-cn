# <a name="unattendedexecutionaccount-property-wmi-msreportserver_configurationsetting"></a>UnattendedExecutionAccount 属性 (WMI MSReportServer_ConfigurationSetting)
  返回以无人参与方式运行报表时报表服务器将模拟的用户帐户。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim UnattendedExecutionAccount As String  
```  
  
```csharp  
public string UnattendedExecutionAccount;  
```  
  
## <a name="property-values"></a>属性值  
 表示帐户名的 `String` 对象。
