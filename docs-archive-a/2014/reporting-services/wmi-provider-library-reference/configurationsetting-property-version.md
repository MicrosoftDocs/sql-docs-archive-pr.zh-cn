# <a name="version-property--wmi-msreportserver_configurationsetting"></a>版本属性 (WMI MSReportServer_ConfigurationSetting)
  以格式 Major.Minor.Build.Revision 返回报表服务器的版本。 只读。  
  
## <a name="syntax"></a>语法  
  
```vb  
Public Dim Version As String  
```  
  
```csharp  
public string Version;  
```  
  
## <a name="property-value"></a>属性值  
 包含报表服务器的版本的 `string`。
