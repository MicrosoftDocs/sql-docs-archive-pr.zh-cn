# <a name="select-the-report-type"></a>选择报表类型
  使用报表向导的此页可以选择表格报表或矩阵报表。  
  
## <a name="options"></a>选项  
 **表格**  
 创建包含表的报表。 表是一种数据按列和行进行排列的数据区域。 表包含一组静态的列，表中的行数取决于数据集中的数据。  
  
 向导中的后续页提供了表定义选项。  
  
 **Matrix**  
 创建包含矩阵的报表。 矩阵是一种数据按列和行进行排列的数据区域。 矩阵所提供的功能与交叉表和透视表的功能类似。 与包括一组静态列的表不同，矩阵的列数可以是动态的。  
  
 向导中的后续页提供了矩阵定义选项。
