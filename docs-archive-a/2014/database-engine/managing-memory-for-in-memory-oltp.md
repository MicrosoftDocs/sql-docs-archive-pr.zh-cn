# <a name="managing-memory-for-in-memory-oltp"></a>管理内存中 OLTP 的内存
  内存优化表要求存在足够的内存，以便将所有行和索引保留在内存中。 因为内存是有限的资源，所以了解和管理系统的内存使用情况非常重要。 本节中的主题介绍一些常见的内存使用和管理方案。
