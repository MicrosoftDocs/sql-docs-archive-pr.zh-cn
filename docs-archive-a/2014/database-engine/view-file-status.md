# <a name="view-file-status"></a>查看文件状态
  源代码管理可以维护每个源代码管理项的详细状态信息。 可以使用此信息标识文件的当前状态，并可用于创建有关项的状态报告。  
  
 状态信息包括：  
  
-   签出文件的用户标识。  
  
-   所选文件的最新版本号。  
  
-   创建版本的日期。  
  
-   与任意版本关联的用户注释。  
  
-   用于共享文件的项目的路径。  
  
### <a name="to-view-the-status-of-an-item"></a>查看项的状态  
  
1.  在解决方案资源管理器中，选择一个项。  
  
2.  在 "**文件**" 菜单上，指向 "**源代码管理**"，然后单击 " **SourceSafe 属性**"。
