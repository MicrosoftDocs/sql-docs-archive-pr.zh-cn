# <a name="friendlyname-property-securitycertificate-class"></a>FriendlyName 属性（SecurityCertificate 类）
  获取或设置安全证书的备选友好名称。  
  
## <a name="syntax"></a>语法  
  
```  
  
object  
.FriendlyName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示安全证书的 [SecurityCertificate 类](securitycertificate-class.md) 对象。  
  
## <a name="property-valuereturn-value"></a>属性值/返回值  
 一个指定安全证书的备选友好名称的字符串值。
