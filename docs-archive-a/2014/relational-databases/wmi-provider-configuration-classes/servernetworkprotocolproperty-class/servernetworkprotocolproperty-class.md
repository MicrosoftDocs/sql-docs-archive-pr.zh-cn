# <a name="servernetworkprotocolproperty-class"></a>ServerNetworkProtocolProperty 类
  [ServerNetworkProtocolProperty Class] ServerNetworkProtocolProperty-class.md) 表示服务器网络协议的属性。  
  
 利用 [ServerNetworkProtocolProperty Class] ServerNetworkProtocolProperty-class.md) ，你可以执行以下任务：  
  
-   检查当前属性所属的协议。  
  
-   检查 IP 地址。  
  
-   检查当前属性的名称、类型和值。  
  
-   修改当前属性的属性字符串值和数值。  
  
-   设置当前属性的标志。
