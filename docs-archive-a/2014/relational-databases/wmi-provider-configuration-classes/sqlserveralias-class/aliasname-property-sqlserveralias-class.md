## <a name="syntax"></a>语法  
  
```  
  
object  
.AliasName [= value]  
```  
  
## <a name="parts"></a>组成部分  
 对象  
 一个表示别名的[SqlServerAlias 类](sqlserveralias-class.md)对象 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。
