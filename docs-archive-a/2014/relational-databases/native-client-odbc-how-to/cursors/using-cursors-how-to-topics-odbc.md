# <a name="using-cursors-how-to-topics-odbc"></a>使用游标操作指南主题 (ODBC)
  若要使用游标，必须首先设置控制 ODBC 游标行为的连接和语句属性。 游标允许应用程序在每次提取时检索多行，并在游标的当前位置执行 UPDATE、INSERT 或 DELETE 语句。
