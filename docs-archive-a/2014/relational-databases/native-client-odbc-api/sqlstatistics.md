# <a name="sqlstatistics"></a>SQLStatistics
  可以对静态游标执行**SQLStatistics** 。 尝试在可更新 (键集驱动或动态) 上执行**SQLStatistics**会返回 SQL_SUCCESS_WITH_INFO，指示游标类型已更改。
