# <a name="export-as-policy-dialog-box"></a>“导出为策略”对话框
  使用此对话框可以将一组基于策略的管理方面属性保存为策略。 可以在 XML 文件中保存策略，也可以将其直接保存到当前 [!INCLUDE[ssDE](../../includes/ssde-md.md)]实例中。  
  
## <a name="options"></a>选项  
 **策略定义名称**  
 键入策略的名称。  
  
 **条件名称**  
 键入条件的名称。  
  
 **目标本地服务器**  
 选择此选项可在本地服务器实例上保存策略。  
  
 **到文件**  
 若要将策略保存为 XML 文件，请键入此文件的路径和名称或者使用“浏览”( **...** ) 按钮。
