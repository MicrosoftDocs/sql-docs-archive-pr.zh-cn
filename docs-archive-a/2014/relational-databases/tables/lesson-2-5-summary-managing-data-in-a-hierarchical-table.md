# <a name="summary-managing-data-in-a-hierarchical-table"></a>摘要：管理层次结构表中的数据
  `hierarchyid` 数据类型使您可以轻松地创建反映分层数据的表，然后再创建深度优先和广度优先的索引以有效地访问此表。 使用分层方法来查询和操作数据。
