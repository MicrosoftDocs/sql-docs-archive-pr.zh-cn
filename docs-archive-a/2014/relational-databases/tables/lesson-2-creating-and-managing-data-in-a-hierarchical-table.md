# <a name="lesson-2-creating-and-managing-data-in-a-hierarchical-table"></a>第 2 课：创建和管理层次结构表中的数据

-   [使用分层方法对层次结构表中的数据重新排序](lesson-2-4-reordering-data-in-a-hierarchical-table-using-hierarchical-methods.md)
