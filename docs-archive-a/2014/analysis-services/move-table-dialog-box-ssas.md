# <a name="move-table-dialog-box-ssas"></a>“移动表”对话框 (SSAS)
  使用“移动表” **** 对话框可以更改表在模型设计器中的显示顺序。 若要访问此对话框，请右键单击要移动的表底部的选项卡，然后选择“移动”****。  
  
## <a name="ui-element-list"></a>UI 元素列表  
 **将所选表移到下表前**  
 您要移动的表将移到从该列表中选择的表之前的位置。 如果想要将某个表移到该组表的末尾，则选择“（移至末尾）”****。
