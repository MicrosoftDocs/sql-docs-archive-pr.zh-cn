# <a name="view-project-properties"></a>视图项目属性
  您可以在“属性”窗口中查看与项目关联的属性。  
  
### <a name="to-view-project-properties"></a>查看项目属性  
  
1.  在解决方案资源管理器中，选择一个项目。  
  
2.  如果未打开“属性”窗口，请单击“视图”  菜单上的“属性窗口”  。  
  
3.  在“属性”窗口中查看项目的属性。  
  
 项目属性（例如创建日期）是只读的，不能从“属性”窗口中更改。
