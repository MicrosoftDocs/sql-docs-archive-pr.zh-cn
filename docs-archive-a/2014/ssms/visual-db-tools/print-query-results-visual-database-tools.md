# <a name="print-query-results-visual-database-tools"></a>打印查询结果 (Visual Database Tools)
  若要打印查询结果，可以将 [“结果”窗格](visual-database-tools.md) 中的内容复制到另一 Windows 程序中，例如字处理程序，然后在其中设置格式并进行打印。  
  
 查询和视图设计器在将查询结果放入剪贴板时，使用制表符作为列之间的分隔符，使用回车符和换行符作为行之间的分隔符。  
  
### <a name="to-print-query-results"></a>打印查询结果  
  
1.  选择要打印的列或行。  
  
2.  使用“复制”  命令将它们移动到剪贴板中。  
  
3.  切换到要用于打印结果的 Windows 程序。  
  
4.  使用“粘贴”  命令从剪贴板中取出查询结果。  
  
5.  设置格式，然后打印结果。
