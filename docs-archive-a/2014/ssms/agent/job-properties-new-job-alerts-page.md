# <a name="job-properties-new-job-alerts-page"></a>作业属性：新建作业（“警报”页）
  使用此页可以查看和组织代理作业的警报 [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 。  
  
## <a name="options"></a>选项  
 **警报列表**  
 列出此作业的警报。  
  
 **添加**  
 为该作业创建新的警报。  
  
 **编辑**  
 修改所选警报的定义。  
  
 **移除**  
 从作业中移除并删除所选警报。
